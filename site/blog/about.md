# About Me

I'm a data Practitioner with 6+ years building production-grade data systems and ML-powered tooling. I enjoy the full stack of data work, from pipeline architecture and database optimization to deploying machine learning models that solve real problems.

## Projects

My own project:

- **[sklx](https://github.com/lazarust/sklx)** — A scikit-learn compatible neural network library that wraps Apple's MLX framework. Inspired by [skorch](https://skorch.readthedocs.io/).

Open source contributions:

- **[mlx-lm](https://github.com/ml-explore/mlx-lm)** — A Python package for generating text and fine-tuning large language models on Apple silicon with MLX.
- **[skops](https://github.com/skops-dev/skops)** — A Python library for sharing scikit-learn models and putting them in production, with integration to Hugging Face Hub.
